# Edwin P. Bayog Jr. - Agentic Portfolio Assistant

This is the repository for the personal portfolio of **Edwin P. Bayog Jr.**, a Computer Engineer specializing in AI, Agentic Development, and IoT.

Designed with a distinct **Neo-Brutalism** aesthetic, this portfolio features an integrated AI conversational assistant to provide interactive insights into Edwin's experience, projects, and skills.

## Features

- **Neo-Brutalism Design System:** Bold typography, high-contrast borders, and unique hover states.
- **Agentic AI Assistant:** Built-in chat interface powered by advanced LLMs (via OpenRouter) to answer questions about Edwin's background and projects.
- **Interactive Global Background:** Thermodynamic mouse-effect background spanning across all sections.
- **Smooth Animations:** Integrated framer-motion and Lenis for fluid, responsive user experience.

## Tech Stack

- **Frontend Framework:** React 19, TypeScript, Vite
- **Styling:** Tailwind CSS v4, shadcn/ui components (Radix UI)
- **Animations:** Framer Motion, Lenis (smooth scrolling), p5.js, Three.js
- **Backend/AI:** Vercel serverless functions (`api/chat.ts`), OpenRouter API, context injection

## Getting Started

To run this project locally:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/naaivvv/portfolio-eb.git
   cd portfolio-eb
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Environment Variables:**
   Create a `.env` file in the root directory and add necessary keys (e.g., OpenRouter API Key for the AI chat functionality).
   ```env
   OPENROUTER_API_KEY=your_api_key_here
   ```

4. **Run the development server:**
   ```bash
   npm run dev
   ```
